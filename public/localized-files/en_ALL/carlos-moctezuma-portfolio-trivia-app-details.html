Trivia App

Project information

  • Category: App design & Optimization
  • Project: DEV.F Fourth Module Final Project
  • Project date: 13 December, 2022
  • Project URL: Trivia App

This is the final project of the fourth module (Advanced JavaScript), the business case required a connection via API to retrieve the questions for the trivia, in addition to count the correct answers and generate various errors in case of not being able to connect to the API via Promises (fetch). The technology stack was:

  • - Async fetch Javascript API
  • - Json Response
  • - Javascript (vanilla)
  • - Git Version control
  • - Firebase deployment
  • - Optimization for search engines (Speed Optimization)
